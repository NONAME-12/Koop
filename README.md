## Быстрый старт

1. Создайте и активируйте виртуальное окружение:
   ```sh
   python -m venv venv
   venv\Scripts\activate  # Windows
   source venv/bin/activate  # Mac/Linux
   ```
2. Установите зависимости:
   ```sh
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```sh
   python main.py
   ```
